## 📝 Описание проекта

Проект содержит автотест для формы на сайте `practice-automation.com`. Тест покрывает следующий сценарий:

1. Заполнение полей Name, Password, Email
2. Выбор чекбоксов Milk и Coffee
3. Выбор цвета Yellow
4. Выбор значения в выпадающем списке "Do you like automation?"
5. Подсчет количества инструментов и поиск самого длинного названия в списке Automation tools
6. Заполнение поля Message с результатами подсчета
7. Отправка формы и проверка появления alert с текстом "Message received!"

При написании автотеста применялся паттерн проектирования Page Object Model и Page Factory

Класс BaseSeleniumPage - абстрактный базовый класс для всех Page Object классов

Класс BaseSeleniumTest содержит настройку ChromeDriver до выполнения теста и закрытие ChromeDriver после теста

Класс MainPage включает в себя поиск элементов на странице и взаимодействие с ними, также отдельно реализован метод перемещения по странице до необходимого элемента

Класс FormFieldsTest - непосредственное исполнение теста

Класс TestListener - реализует формирование скриншота для отчета Allure, в случае падения теста

## 🚀 Запуск теста и генерация отчета Allure

Чтобы заупстить автотест следует выполнить команду "./gradlew clean test" - проект пересоберется и запустит тест

Чтобы сгенерировать отчет Allure следует выполнить команду "./gradlew allureServe" - отчет автоматически откроется в браузере

## ✅ Позитивный тест-кейс по веб-форме

### Покрытие нестандартных форматов данных

Предусловие: открыта веб-форма по адресу https://practice-automation.com/form-fields/

№ | Шаг | Ожидаемый результат
|-|------------------------|--------|
1 |	В поле Name ввести: test name | Поле принимает значение
2 |	В поле Password ввести: 0000 | Поле принимает значение
3 |	В списке "What is your favorite drink?" выбрать: Ctrl-Alt-Delight | Чекбоксы Ctrl-Alt-Delight отмечен
4 | В списке "What is your favorite color?" выбрать: #FFC0CB | Чекбокс #FFC0CB отмечен
5 | В дропдауне "Do you like automation?" выбрать: Undecided | В дропдауне выбран вариант Undecided 
6 |	В поле Email ввести test+.test@example.com.com | Поле принимает значение
7 |	В поле Message ввести строку: hello, world!5	| Поле принимает значение
8 |	Нажать Submit	| Появляется alert с текстом "Message received!"

## ⚠️ Негативный тест-кейс по веб-форме

### SQL-инъекция через поле Message

Предусловие: открыта веб-форма по адресу https://practice-automation.com/form-fields/

№ | Шаг | Ожидаемый результат
|-|------------------------|--------|
1 |	В поле Name ввести: Test | Поле принимает значение
2 |	В поле Password ввести: 123 | Поле принимает значение
3 |	В списке "What is your favorite drink?" выбрать: Coffee, Milk | Чекбоксы Coffee и Milk отмечен
4 | В списке "What is your favorite color?" выбрать: Yellow | Чекбокс Yellow отмечен
5 | В дропдауне "Do you like automation?" выбрать: Yes | В дропдауне выбран вариант Yes 
6 |	В поле Email ввести test@example.com | Поле принимает значение
7 |	В поле Message ввести строку: '; DROP TABLE users; --	| Поле принимает значение
8 |	Нажать Submit	| Форма не должна отправляться (появляется ошибка валидации) или, если форма отправляется, alert с текстом "Message received!" не должен появляться
